Política de Privacidade

Última atualização:

1. Quem somos

Este site ("Agendamento Aurora") é um aplicativo cliente que permite ao usuário agendar horários e criar eventos no seu próprio Google Calendar para chamadas via WhatsApp.

2. Dados que tratamos

  • Nome completo e grau de parentesco informados pelo próprio usuário;
  • Data e horário escolhidos para o agendamento;
  • Dados mínimos de perfil do Google exigidos pelo processo de autorização (OAuth) somente para criar eventos no calendário do usuário.

3. Como usamos os dados

Os dados são utilizados exclusivamente para criar o evento no Google Calendar do próprio usuário com lembretes opcionais e para montar a mensagem de confirmação no WhatsApp. Não armazenamos os dados em banco de dados próprio ou servidor.

4. Compartilhamento

Não compartilhamos dados com terceiros. O processamento ocorre localmente no navegador do usuário e através das APIs oficiais do Google, respeitando os termos do Google API Services.

5. Segurança

Utilizamos OAuth 2.0 para autorização segura. O acesso é limitado ao escopo https://www.googleapis.com/auth/calendar.events, necessário para criar e listar eventos.

6. Direitos do usuário

Você pode revogar a autorização a qualquer momento acessando Suas conexões de conta Google.

7. Contato

Para dúvidas ou solicitações, entre em contato pelo e‑mail indicado no aplicativo.
